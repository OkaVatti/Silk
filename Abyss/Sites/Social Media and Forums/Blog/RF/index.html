Silk

basically what you're looking at

Silk is basically the entire source code of this website. It contains 95% of everything that is on this website, excluding The Gateway, secrets, and the PHP code which has yet to be added to the website. Silk is no more than just a little box for me to maintain, organize, and deploy whenever I need. Say I eventually have to switch server hostin providers or, worst case senario, I have to move everything to my own servers. I can easily deploy my website within a matter of seconds. Making updates extremely fast an easy. The only downside to this is I'm not sure how well this will scale.
